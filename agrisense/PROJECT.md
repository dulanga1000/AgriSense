## 1. Introduction

AgriSense is a smart agriculture mobile application developed using Flutter. It provides farmers with tools to detect crop diseases, access real-time weather data, and receive intelligent farming recommendations.

---

## 2. Objectives

- Detect crop diseases using AI  
- Provide weather-based crop recommendations  
- Improve agricultural productivity  
- Offer an easy-to-use mobile solution  

---

## 3. Problem Statement

Farmers face several challenges:

- Difficulty identifying crop diseases early  
- Lack of access to expert advice  
- Limited understanding of weather impacts  

These issues lead to crop damage and reduced productivity. AgriSense solves this using mobile and AI technologies.
